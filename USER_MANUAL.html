Welcome! This guide explains every feature in plain language and shows a sample file for each procedure.

Quick Start

  1. Click Data → Open CSV/XLSX… and load a file.
  2. Go to Analyze tab. Pick a Procedure.
  3. Choose columns: Y, X, Group, or list Predictors (comma/space separated; quotes allowed).
  4. Press Run. Results appear on the right. Export from Results & Export.

Tip: Open the matching sample data from samples/ to try each method.

Which columns go where?

  • Y / Outcome: the main thing you’re measuring (numeric for most tests).
  • X / Second Var: a second measurement (paired tests, ROC score) or variance column for meta-analysis.
  • Group: a categorical label (e.g., A/B/C) defining groups.
  • Predictors: one or more columns used by the model (separate by comma/space). Quotes are OK for names with spaces.

Procedures & Sample Files

Below, each method tells you what it asks, why you’d use it, how to run it here, and the sample file to practice.

Crosstab (Chi-square / Fisher)

What: Tests association between two categories. Why: Are treatment and outcome related?

How: Y=Outcome, X=Treatment. Check Fisher for 2×2 small samples.

Sample: samples/sample_crosstab.csv

T-tests & Nonparametrics

  • One-sample t: Is one mean different from μ? Y=Score, Constant=50. File: sample_ttest_one_sample.csv
  • Independent t: Two groups (A,B). Y=DV, Group=Group, Levels=A,B. File: sample_ttest_independent.csv
  • Paired t: Before vs After. Y=Pre, X=Post. File: sample_ttest_paired.csv
  • Mann–Whitney U: Like independent t for ranks. Use the independent t sample.
  • Wilcoxon signed-rank: Like paired t for ranks. Use the paired t sample.

ANOVA Family

  • One-way ANOVA / Tukey / Kruskal: Y=Y, Group=Group. File: sample_anova_oneway.csv
  • Repeated-measures ANOVA: Y=score, Subject=subject, Within=time. File: sample_anova_repeated_long.csv
  • ANCOVA: Y=Y, Group=Group, Predictors=Cov1 Cov2. File: sample_ancova.csv
  • Friedman: Predictors=m1 m2 m3. File: sample_friedman.csv

Correlations

Predictors=x y z. File: sample_correlations.csv

Regression & GLMs

  • OLS: Y=Y, Predictors=X1 X2. File: sample_ols.csv
  • Logistic: Y=Y, Predictors=X1 X2. File: sample_logistic.csv
  • Multinomial: Y=Y, Predictors=X1 X2. File: sample_multinomial.csv
  • Ordinal: Y=Y (1–5), Predictors=X1. File: sample_ordinal.csv
  • Poisson: Y=Y (counts), Predictors=X. File: sample_glm_poisson.csv
  • Negative binomial: Y=Y, Predictors=X. File: sample_glm_negbin.csv
  • Gamma: Y=Y (>0), Predictors=X. File: sample_glm_gamma.csv
  • Linear mixed model: Y=Y, Predictors=X, Cluster=Cluster. File: sample_lmm.csv

Reliability & Agreement

  • Cronbach α: Predictors=Item1 Item2 Item3 Item4 Item5. File: sample_cronbach_alpha.csv
  • Cohen κ: Y=RaterA, X=RaterB. File: sample_kappa.csv
  • Weighted κ (linear/quadratic): same as above; ratings should be ordered.
  • Fleiss κ (multi-rater): Predictors=R1 R2 R3. File: sample_fleiss_icc_kripp.csv
  • ICC: Predictors=R1 R2 R3. File: sample_fleiss_icc_kripp.csv
  • Krippendorff α (nominal/ordinal/interval): Use Predictors or Y & X for exactly two raters. File: sample_fleiss_icc_kripp.csv

Survival & Diagnostics

  • Survival (KM + Cox + Weibull AFT): Time=time, Event=event, Group=group, Predictors=age. File: sample_survival.csv
  • ROC / PR / Brier: Y=Y (0/1), X=Score. File: sample_roc_pr.csv

Time Series

  • ARIMA/ETS: Time=Date, Y=Value. File: sample_timeseries.csv

Meta-analysis

Y=yi (effect), X=vi (variance), Group=by (optional). File: sample_meta_dl.csv

Causal Inference

  • IPTW (ATE): Y=Y (0/1), Group=A (0/1), Predictors=X1 X2. File: sample_iptw.csv
  • Difference-in-Differences: Y=Y, Group=Treat, Second Group=Post. File: sample_did.csv

Dimension Reduction & Clustering

  • PCA: Predictors=f1 f2 f3 f4. File: sample_cluster_pca.csv
  • EFA: Predictors=Item1..Item6. File: sample_efa.csv
  • Correspondence analysis: Group=Brand, Second Group=Preference. File: sample_correspondence.csv
  • K-Means / Agglomerative / Auto-KMeans: Predictors=f1 f2 f3 f4. File: sample_cluster_pca.csv

Classification (LDA / Tree / CHAID-like / RF)

Y=Y, Predictors=X1 X2. File: sample_classifiers.csv
